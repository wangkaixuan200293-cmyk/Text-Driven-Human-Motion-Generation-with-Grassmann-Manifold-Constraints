# Text-Driven-Human-Motion-Generation-with-Grassmann-Manifold-Constraints
We propose a geometry-aware motion generation method with Grassmannian constraints. 
Manifold constraints are used in VAE and diffusion with a timestep-adaptive strategy. 
Our method achieved an 18.4% reduction in FID compared to baseline models on HumanML3D and KIT-ML. This work fills the gap in manifold modeling and provides a reusable paradigm.

Full implementation code will be open-sourced after the formal acceptance of the corresponding paper.
